Coins
===

Given a set of coin values *S* calculate the minimum number of coins needed to make a value of *K*. You can use each coin as many times as you like.

Input
===
```
S = {5, 2, 1}
K = 8
```

Output
===
```
3
```
Explanation: (1x5 + 1x2 + 1x1)

Input
===
```
S = {5, 2, 1}
K = 18
```

Output
===
```
5
```
Explanation: (3x5 + 1x2 + 1x1)

Input
===
```
S = {1, 15, 25}
K = 30
```

Output
===
```
2
```
Explanation: 2x15
